Psycopg is the most popular PostgreSQL adapter for the Python programming language. At its core it is a complete implementation of the Python DB API 2.0 specifications. Several extensions allow access to many of the features offered by PostgreSQL.

Psycopg is released under the terms of the GNU Lesser General Public License, allowing use from both free and proprietary software.

The psycopg3 project

psycopg3 is the new implementation of the most used, reliable, feature-rich PostgreSQL adapter for the Python programming language.

A familiar DBAPI interface, or an asyncio-based asynchronous interface

Asynchronous communication with Python and with the database

Rich and flexible adaptation system, optionally using binary format for efficient data transfer

High performance data loading using COPY directly from Python objects

psycopg3 design emerges from the experience of more than 10 years of development and support of psycopg2, embracing and addressing the new possibilities offered by the new generations of the language and the database and the emerging patterns in software development and deployment.

Project funding

psycopg3 is current in development phase. A few project milestones have been documented in a series of blog articles. Full-time development will start in October 2020.

Part of the project is already written, but now the code needs to grow the several features useful for a seamless interaction between the database and the language.

Funding the project would allow full-time work on the database and the initial release in a few months. If you are interested in sponsoring the development you are welcome to contribute.

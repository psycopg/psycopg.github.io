Psycopg 3 – PostgreSQL database adapter for Python¶

psycopg3 is a newly designed PostgreSQL database adapter for the Python programming language.

psycopg3 presents a familiar interface for everyone who has used psycopg2 or any other DB-API 2.0 database adapter, but allows to use more modern PostgreSQL and Python features, such as:

  • Strict Strong Typing

  • asynchronous support

  • server-side parameters binding

  • binary communication

  • a great integration of the COPY support

  • direct access to the libpq functionalities
